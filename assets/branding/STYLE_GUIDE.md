## 3. TIPOGRAFIA

### 3.1 Filosofia tipografica
Titulos masivos y audaces que transmiten confianza, con una jerarquia precisa y pesos
bien definidos. El resultado: tipografia que se siente creativa y profesional al mismo
tiempo — como la voz de un amigo experto que habla con claridad.

### 3.2 Familia principal: Plus Jakarta Sans
Fuente geometrica moderna, amigable, con personalidad.
Google Fonts: `Plus+Jakarta+Sans:wght@300;400;500;600;700;800`

**Por que Plus Jakarta Sans:**
- Geometrica pero no fria (a diferencia de Inter o Helvetica)
- Soporta pesos de 200 a 800 (rango completo para jerarquia)
- Optima para digital, excelente legibilidad
- Feeling moderno/startup que conecta con millennials
- Gratuita y altamente optimizada para web

### 3.3 Familia de display (titulos hero): Space Grotesk
Para titulos oversized y statements de impacto.
Google Fonts: `Space+Grotesk:wght@400;500;600;700`

**Por que Space Grotesk:**
- Personalidad unica — no generica como Inter o Roboto
- Letras con detalles geometricos que la hacen memorable
- Perfecta para titulos grandes (60px+)
- Contrasta con Plus Jakarta creando dinamismo visual

### 3.5 Reglas tipograficas
- Titulos hero SIEMPRE en Space Grotesk Bold
- Maximo 2 lineas en hero titles (impacto > explicacion)
- Body text nunca menor a 16px en desktop
- Links en texto: peso 600, color primario, sin subrayado (underline on hover)
- No usar italicas excepto en citas textuales
- Numeros y metricas: Space Grotesk Medium, tamano display

## 8. ESTILO DE ILUSTRACION

### 8.1 Filosofia visual
Ilustraciones con estetica painterly/vintage que crean un contraste rico contra
la tipografia moderna y bold. Tematica mexicana acuatica (Xochimilco, axolotls,
lirios, naturaleza mexicana). El estilo evoca un mural mexicano moderno —
contemporaneo pero con raiz profunda.

### 8.2 Estilo artistico

```
TIPO:           Ilustracion digital con textura painterly (no flat, no 3D)
ESTETICA:       Vintage contemporaneo — mural mexicano moderno con precision digital
TEXTURAS:       Pinceladas visibles, grano sutil, capas de color traslucidas
PALETTE:        Azules profundos, turquesas, corales, dorados, verdes naturales
REFERENCIA:     Muralismo mexicano moderno + botanica acuatica de Xochimilco
LUZ:            Calida, como luz de atardecer sobre agua
```

### 8.3 Tematica de las ilustraciones

```
HERO PRINCIPAL
  Escena:   Vista acuatica de Xochimilco — agua azul profundo con
            lirios, vegetacion, AXO nadando entre elementos
  Overlay:  Gradiente oscuro de izq a derecha para legibilidad de texto
  Efecto:   Texto hero en blanco sobre la ilustracion

SECCIONES DE FEATURES
  Escena:   Detalles botanicos — hojas, flores, agua — enmarcando
            el contenido con elementos decorativos en esquinas
  Estilo:   Mas sutil, como bordes ilustrados

SECCION DE IMPACTO (10%)
  Escena:   AXO sembrando semillas que crecen en diferentes formas
            (tiendas, familias, comunidades)
  Palette:  Tonos calidos, dorados, verdes

SECCION DE AI / TECNOLOGIA
  Escena:   Vista nocturna/cosmica — cielo de Oaxaca — con AXO
            flotando entre constelaciones de datos
  Palette:  Azul profundo, violeta, puntos brillantes

SECCION TESTIMONIALS
  Escena:   Marco ilustrado floral/botanico alrededor de las cards
  Estilo:   Decorativo pero no intrusivo
```

### 8.4 AXO — Mascota ilustrada

```
ESTILO:         Mismo estilo painterly que las ilustraciones de fondo
NO ES:          Cartoon flat, 3D render, emoji, pixelart
ES:             Ilustracion artistica con personalidad, como un personaje
                de un libro ilustrado premium

ESTADOS VISUALES:
  Default:      AXO nadando tranquilo, branquias rosa coral
  Celebrando:   Branquias doradas, burbujas a su alrededor
  Pensando:     Branquias azul suave, mirada hacia arriba
  Guiando:      AXO senalando con la cola hacia un elemento
  Impactando:   AXO con halo de luz, sembrando

TAMANOS:
  Hero:         400-600px (integrado en la ilustracion de fondo)
  Inline:       120-200px (acompanando secciones)
  Icono:        48-64px (versiones simplificadas para badges)
  Micro:        24-32px (chat, notificaciones, avatares)

COLORES AXO:
  Cuerpo:       Rosa coral #F5A0A0 a #E8B4B4
  Branquias:    Coral vibrante #FF7B7B (default), cambia segun estado
  Ojos:         Negro #1A1A1A con reflejo blanco
  Manchas:      Dorado suave #F5D5A0 (sutiles)
```

### 8.5 Reglas para ilustraciones

```
DO:
  ✓ Usar texturas de pincelada visibles
  ✓ Mantener la palette de color de la marca
  ✓ Integrar elementos mexicanos naturales (xochimilco, flora nativa)
  ✓ Crear profundidad con capas de transparencia
  ✓ Dejar espacio claro para texto (no ilustrar encima del copy)
  ✓ Usar ilustraciones como fondo de secciones hero

DON'T:
  ✗ Usar stock illustrations genericas
  ✗ Mezclar estilos (flat icons + painterly = inconsistente)
  ✗ Hacer ilustraciones demasiado detalladas que compitan con el contenido
  ✗ Usar colores neon o fuera de la paleta
  ✗ Hacer a AXO demasiado cartoon o infantil
  ✗ Saturar de elementos — menos es mas
```


## 12. MOTION & ANIMACION

### 12.1 Principios
- Movimiento sutil y con proposito (nunca decorativo)
- Respeta `prefers-reduced-motion`
- Duraciones cortas: 200-300ms para micro-interactions, 500-1000ms para entradas

### 12.2 Curvas de animacion

```
--ease-default:    cubic-bezier(0.25, 0.1, 0.25, 1)     → General
--ease-spring:     cubic-bezier(0.34, 1.56, 0.64, 1)    → Botones, bouncy
--ease-smooth:     cubic-bezier(0.45, 0, 0.55, 1)       → Entradas de seccion
--ease-out:        cubic-bezier(0, 0, 0.2, 1)            → Salidas
```

### 12.3 Animaciones del sitio

```
FADE IN UP (Secciones)
  De:       opacity 0, translateY(20px)
  A:        opacity 1, translateY(0)
  Duracion: 600ms
  Trigger:  Al entrar en viewport (IntersectionObserver)
  Delay:    Escalonado 100ms entre elementos hermanos

SCALE IN (Cards)
  De:       opacity 0, scale(0.95)
  A:        opacity 1, scale(1)
  Duracion: 400ms

TEXT REVEAL (Titulos hero)
  Tipo:     Line-by-line mask reveal
  De:       Cada linea translateY(100%), opacity 0
  A:        translateY(0), opacity 1
  Duracion: 800ms por linea
  Delay:    200ms entre lineas
  Herramienta: GSAP SplitText o CSS clip-path

COUNTER UP (Stats/metricas)
  Tipo:     Conteo numerico animado
  Duracion: 1500ms
  Easing:   ease-out
  Trigger:  Al entrar en viewport

PARALLAX SUTIL (Ilustraciones de fondo)
  Intensidad: 0.1-0.2 (muy sutil)
  Uso:        Solo en secciones hero con ilustracion
```
